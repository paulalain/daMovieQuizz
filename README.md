# Da Movie Quizz
Test your movie knowledge.

========

Prerequisites:

  * Node.js NPM
  * Bower (install `npm install -g bower`)
  * Yeoman (install `npm install -g generator-backbone`)
  * Sass (install 'gem install sass')
  * Grunt (install: `npm install -g grunt-cli`)

Install:
 * `npm install`
 * `bower install`
 * `grunt build`

Launch server:

 * `npm install`
 * `bower install`
 * `grunt serve`

Tests:
	* `cd test`
	* `bower install`
	* `grunt test`
