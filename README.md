Da Movie Quizz
========

Prerequisites:

  * NPM
  * Bower (install `npm install -g bower`)
  * Yeoman (install `npm install -g generator-backbone`)
  * Sass (install 'gem install sass')
  * Grunt (install: `npm install -g grunt-cli`)


Install:
 * `npm install`
 * `bower install`
 * `grunt build`
 * copy dist directory to server

Test:
	
	* `grunt test`
